# Mark00

> 作为第一个初始作品，这个项目的作用在于其入门性和引领性。

## 1. Requirements

硬件层面：

- 可以通过蓝牙、WIFI完成升级和部分APP应用。
- 拥有可以触控的屏幕。
- 有温度、湿度传感器。
- 可以读取SD。
- 能有电池，且续航很好。
- 可以发出声音。

软件层面：

- 结合多端完成TODO显示
- 能作为桌面显示，并多端控制。
- 信息显示。
- 充放电动效。
- SD卡自动升级。

后期增强：

- 太阳能充电
